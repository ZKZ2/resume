Projects

  • RSA and DES Encryption Using Altera FPGA (Fall 2021)

    With current demand for data privacy, we wanted to explore the use of cryptography for data
    transmission. We designed a secure messaging platform that utilized a pair of programmable FPGA
    controllers with custom code to implement industry standard RSA/DES encryption.

  • Distance Calculator (Spring 2022)

    Health authorities mandated that a certain distance must be maintained between people to minimize
    spread of disease. We designed a distance measurement device for people in queues to ensure these
    guidelines were met. The major design goal was to create an affordable solution that would be widely
    available and rapidly deployable. Using off the shelf products and a simple feedback mechanism to alert
    people when they are approaching the safe distance limit.

  • Battleship (Fall 2022 - Spring 2023)

    Team lead of software development team working to redesign the classic game of battleship.
    The goal was to develop a prototype that would provide more interactivity for players, such as a
    hardware touch interface and multi color ship markers. We had to work closely with the hardware team
    to ensure proper integration between the software and the hardware components.

  • RFID Attendance Tracker using 2FA (Spring 2023)

    Wide spread use of RFID tags and the availability of tools to clone them is a major issue. We wanted to
    design and build a proof-of-concept project to prevent unauthorized usage of RFID tags in the attendance
    tracker. The system implemented two-factor authentication (a passcode prompt) as added protection against
    clone tags.
